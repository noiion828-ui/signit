## Ausfuehrliche Beschreibung (max. 16.000 Zeichen)

```
SignIt — Deine Stimme. Deine Daten. Keine Ausreden mehr.

Du willst eine Petition unterschreiben — aber das Formular nervt. Name tippen, Email tippen, PLZ, Stadt, Strasse. Und dann noch schnell das Bestaetigungs-Email suchen. Jedes Mal. Fuer jede Petition.

SignIt macht das in Sekunden.


WAS SIGNIT MACHT

Autofill in einem Klick
Besuche eine Petition auf openPetition.de. SignIt erkennt das Formular automatisch und fuellt es aus — Vor- + Nachname, Email, PLZ, Stadt. Du klickst nur noch auf "Unterschreiben".

Mehrstufige Formulare kein Problem
openPetition nutzt ein mehrstufiges AJAX-Formular: Step 1 (Name + Email), Step 2 (Adresse, nach reCAPTCHA injiziert). SignIt erkennt beide Schritte automatisch per MutationObserver und fuellt auch Step 2 aus, sobald er erscheint.

Gmail / Outlook Deeplink
Nach dem Unterschreiben brauchst du das Bestaetigungs-Email von openPetition. SignIt zeigt dir einen Toast mit direktem Link zu deinem Posteingang — Gmail oder Outlook — damit du keine Stimme verlierst.

Impact Tracking
SignIt merkt sich, welche Petitionen du unterschrieben hast, und prueft alle 4 Stunden die aktuellen Unterschriften-Zahlen. Im Popup siehst du: wie viele neue Unterschriften seit deiner Stimme gesammelt wurden. Du weisst, dass deine Stimme den Anstoss gegeben hat.

Badge mit Echtzeit-Update
Die Zahl neuer Unterschriften erscheint als gruenes Badge auf dem Extension-Icon — auch wenn du das Popup nicht oeffnest.


WIE ES FUNKTIONIERT

1. Extension installieren
2. Popup oeffnen → Profil einmal anlegen (Name, Email, PLZ, Stadt, Strasse optional)
3. Eine Petition auf openpetition.de besuchen
4. SignIt fuellt das Formular automatisch aus
5. Unterschreiben, Bestaetigungs-Email bestaetigen
6. Im Popup den Petitions-Tab oeffnen: Impact-Counter laeuft


DATENSCHUTZ — 100% LOKAL

Alle deine Daten (Name, Email, Adresse) werden ausschliesslich lokal in chrome.storage.local gespeichert. Kein Server. Keine Datenbank. Keine Telemetrie. Kein Account erforderlich.

SignIt liest Petitionsseiten nur zum Zaehlen von Unterschriften (direkter Fetch der oeffentlichen Seite). Es werden keine personenbezogenen Daten uebertragen.

Berechtigungen:
- storage: lokale Speicherung deines Profils und deiner Petitions-Liste
- activeTab: Zugriff auf die aktive Petitionsseite zum Autofill
- alarms: periodisches Impact-Polling (alle 4 Stunden)
- Host Permission fuer openpetition.de: Autofill und Unterschriften-Zaehlung

Keine weiteren Berechtigungen. Kein "all sites" Zugriff.


KOMPATIBILITAET

- openPetition.de (vollstaendig unterstuetzt: Step 1 + Step 2 + Confirmation Toast)
- Weitere Petitionsplattformen folgen in zukuenftigen Versionen


FUER WEN IST SIGNIT?

Fuer alle, die regelmaessig Petitionen unterschreiben und keine Zeit mit repetitiven Formularen verschwenden wollen. Aktivisten, Engagierte, neugierige Buergerinnen und Buerger — jeder, dem es wichtig ist, schnell und zuverlaessig zu unterschreiben.


OPEN SOURCE

https://github.com/noiion/signit


KONTAKT & SUPPORT

Webseite: https://signit.tools
Probleme oder Feedback? Issues auf GitHub oder via signit.tools.
```
